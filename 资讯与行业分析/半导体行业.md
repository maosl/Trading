# 一、半导体产业是什么？
半导体 = 芯片 + 制造它的整个产业链
核心作用：**所有电子产品的“大脑”和“心脏”**，负责计算、控制、存储、信号处理。

整个产业链可以简单分成 4 大块：

## 1. 上游：材料 & 设备
- 材料：硅片、光刻胶、特种气体、靶材、湿电子化学品等
- 设备：光刻机、刻蚀机、薄膜沉积、清洗、检测设备
作用：造芯片必须的“工具和原料”

## 2. 中游：芯片设计（Fabless）
代表：华为海思、高通、英伟达、AMD、联发科、卓胜微、兆易创新等
工作：设计电路逻辑，输出 GDS 版图

## 3. 中游：芯片制造（Foundry 晶圆厂）
代表：台积电、中芯国际、华虹半导体
工作：把设计好的图纸在硅片上“雕刻”出芯片

## 4. 下游：封测 & 模组
封测：长电科技、通富微电、华天科技
模组：传感器模组、射频模组、功率模组
作用：把裸片封装、测试，变成可用的芯片

# 二、半导体主要芯片类型（最关键）
## 1. 逻辑芯片（负责“思考计算”）
- CPU / GPU / NPU：电脑、服务器、AI算力
- MCU 单片机：家电、汽车、工控、物联网
- FPGA：通信基站、AI加速、工业控制
- 基带芯片：手机 5G/4G 通信

## 2. 存储芯片（负责“记东西”）
- DRAM：内存（手机、电脑、服务器）
- NAND Flash：固态硬盘 U盘、手机存储
- NOR Flash：工控、汽车、IoT 启动存储

## 3. 模拟芯片（负责“电源&信号”）
- 电源管理芯片 PMIC：几乎所有电子设备必备
- 信号链芯片（运放、ADC/DAC）：音频、图像、测量
- 接口芯片：USB、HDMI、网口

## 4. 功率半导体（负责“管电”）
- MOSFET、IGBT、SiC、GaN
用途：电源转换、电机驱动、快充、新能源

## 5. 射频半导体（负责“无线通信”）
- PA、LNA、射频开关、滤波器
用途：手机、基站、WiFi、蓝牙

## 6. 传感器芯片（负责“感知世界”）
- CIS 图像传感器：手机摄像头、车载摄像头
- MEMS：加速度计、陀螺仪、麦克风
- 指纹、压力、电流、温度传感器

# 三、半导体最终应用在哪些产品？（按景气度排序）

## 1. AI 服务器 / 数据中心（当前最强增长）
用到：
- GPU、AI 加速芯片
- CPU、FPGA
- 高速接口芯片、存储芯片
- 光模块配套芯片、电源管理芯片

对应产品：
英伟达 H100/H200、AMD MI300、国产训练芯片、服务器整机

## 2. 智能手机（最大消费电子市场）
用到：
- 应用处理器（AP）
- 基带、射频芯片
- CIS 图像传感器
- 存储（DRAM+NAND）
- 触控、指纹、音频、电源管理

## 3. 新能源汽车（半导体用量爆发式增长）
一辆车 = 传统车 3～5 倍芯片量
用到：
- IGBT / SiC 功率器件（三电核心）
- 车规 MCU
- 自动驾驶芯片（地平线、英伟达、Mobileye）
- 车载 CIS、激光雷达芯片
- 存储、通信、座舱芯片

## 4. 光伏 / 储能 / 充电桩（能源革命核心）
用到：
- IGBT、SiC 模块
- 驱动芯片、采样芯片
- 主控 MCU、电源管理

## 5. 通信基站与网络设备（5G/6G 基建）
用到：
- 射频 PA、滤波器
- FPGA、高速接口芯片
- 光通信芯片、时钟芯片

## 6. 工业控制 & 机器人
用到：
- 工控 MCU、DSP
- 伺服驱动芯片
- 工业传感器、隔离芯片

## 7. 家电 & 消费电子
电视、空调、冰箱、洗衣机、扫地机、TWS、手表
主要用：
- 低端 MCU
- 电源管理
- 驱动芯片
- 传感器

## 8. 医疗、安防、航空航天
- 医疗：影像芯片、高精度模拟芯片
- 安防：IPC 芯片、NVR 存储
- 航天军工：抗辐照芯片、高可靠器件
